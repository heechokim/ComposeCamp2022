#### Compose 란?
- 컴포즈는 Jetpack 라이브러리에 속함
- 새로운 안드로이드 UI 툴킷
- 이제 XML 레이아웃? 필요없음
- XML 레이아웃 만드는 대신, composable 함수 호출만 하면 됨
- 컴포즈 컴파일러가 컴파일해서 UI 만들어줌

#### Composable 함수란 ?

- 컴포저블 함수를 선언하고 호출함으로써 UI를 만들 수 있음
- 선언형 UI

#### 안드로이드 스튜디오의 preview 기능 사용하기

- `@Preview` 어노테이션
- `@Composable` 어노테이션 보다 먼저 붙여야 함
- 앱 빌드 안해도 미리 볼 수 있다
- 단, 파라미터 없는 composable 함수에서만 동작
- <img width="453" alt="image" src="https://user-images.githubusercontent.com/31889335/200331165-574531d9-30df-49c7-a16b-bfc38cbf4df9.png">
- MessageCard()는 파라미터 있기 때문에 @Preview 못 붙임
- `PreviewMessageCard()` 라는 파라미터 없는 함수로 한 번 감싸서 @Preview 붙이면 됨
- Rebuild 한번 해주기
  - 이유는 PreviewMessageCard()는 만들어두기만 했지 호출하는 곳은 없었음
  - 리빌드 한 번 해주면 안드로이드 스튜디오가 인식하고
  - preview 창에 UI 보여준다
- 리프레쉬 버튼 클릭해주기
  - 안드로이드 스튜디오가 인식한 preview 함수에 변경 사항이 생기면
  - 리프레시 버튼 클릭해주면 업데이트 된다
